Employee Database

This page displays a list of all active employees.

Clicking the More Info button reveals the employee's details, where the Admin can edit or delete their information.

Clicking the Delete button prompts a confirmation box as a safeguard in case the Admin clicked it by mistake.

At the top right corner of the page, the Add Employee button opens a form to add a new employee record. If either of the two text fields is left empty, a warning message will appear, and no data will be submitted to the system.
